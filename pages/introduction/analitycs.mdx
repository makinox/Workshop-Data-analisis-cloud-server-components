# Introducción al analisis de datos

El análisis de datos es el proceso de inspeccionar, limpiar, transformar y modelar datos con el objetivo de descubrir información útil, llegar a conclusiones y respaldar la toma de decisiones informada.

En la actualidad, se genera una gran cantidad de datos en diversas formas y fuentes, como transacciones comerciales, registros médicos, interacciones en redes sociales y sensores de dispositivos conectados, entre otros.

En el ejemplo de hoy veremos como utilizar estos datos para ver un poco mas claramente el rendimiento de unos equipos de futbol de ejemplo.
